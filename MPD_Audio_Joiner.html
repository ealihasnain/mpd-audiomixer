02 · PROCESSING
Pipeline options
Each stage runs in this order: trim silence → join with crossfade → breath removal → loudness normalization. Click any card to toggle. Breath removal is marked experimental — verify result with A/B playback before export.
Join with crossfade
Sample-perfect concatenation. Crossfade smooths the boundary between parts so the split is inaudible.
8 ms
0 ms = hard cut · 8 ms = MPD default · 20+ ms = audible fade
Trim silence
Removes leading and trailing silence from each clip before joining. Hysteresis-based detection preserves soft syllables.
−45 dB
−45 dB = conservative · −35 dB = aggressive (may clip breaths)
80 ms
Tail silence kept before hard cut — avoids abrupt endings
Breath / click removal
Spectral-ratio detection: attenuates segments where low-frequency energy (80–400 Hz) dominates over mid (1–4 kHz). Can eat soft consonants if too aggressive. Preview before exporting.
3 / 10
1 = only obvious breaths · 5 = moderate · 10 = aggressive (DO NOT use above 5 without A/B check)
−18 dB
How much to duck detected breaths. −18 dB is natural. −40 dB silences completely (often artificial)
EBU R128 loudness
Gated ITU-R BS.1770-4 loudness normalization with K-weighted filters and absolute/relative gates. True-peak-limited output to prevent inter-sample clipping.
